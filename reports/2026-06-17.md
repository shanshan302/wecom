# 企业微信 AI / Agent 情报日报 - 2026-06-17

## 今日结论

暂无看到企业微信面向普通企业全面开放“以成员身份任意主动发送消息”的通用 API。但 AI / Agent 方向出现了更明确的工具化路径：企业微信团队维护的 `wecom-cli` 已把消息、文档、智能表格、会议、日程、待办等能力包装为可被 Agent 调用的 CLI / Skills；OpenClaw 插件也显示智能机器人正在成为 Agent 接入企微工作流的核心入口。

判断：近期更可能继续通过“API 模式智能机器人、OpenClaw、CLI / Skills、腾讯云 ADP Connector”逐步放开受控能力，而不是直接开放裸的用户代发消息 API。

## 官方与高可信更新

- [官方] WeComTeam `wecom-cli` 仍是最值得盯的官方信号源。README 显示它定位为“让人类和 AI Agent 都能在终端中操作企业微信”，能力覆盖文档、智能表格、消息、通讯录、待办、会议、日程。消息能力包括会话列表、消息记录拉取、多媒体下载和发送文本。来源：https://github.com/WecomTeam/wecom-cli
- [官方] `wecom-cli` 的能力边界按企业规模区分：10 人以上企业场景强调文档、智能表格、智能文档 CLI 能力；10 人及以下个人/小团队场景覆盖消息、文档、日程、会议、待办等能力。这个限制对“消息 API 是否进一步开放”很关键。来源：https://github.com/WecomTeam/wecom-cli
- [官方] `wecom-cli` 的 Skills 文档列出了 `wecomcli-msg`，包含查询会话、查询消息记录、下载媒体文件、发送文本消息。来源：https://github.com/WecomTeam/wecom-cli/blob/main/docs/skills.md
- [官方] WeComTeam `wecom-openclaw-plugin` 显示企业微信团队正在维护 OpenClaw 官方企业微信插件，支持 Bot mode 与 Agent mode，并描述了私聊、群聊、流式回复、主动消息等能力。来源：https://github.com/WecomTeam/wecom-openclaw-plugin

## AI / Agent / MCP / 智能机器人动态

- [半官方/生态] 腾讯云开发者社区 2026-06-01 发布 ADP 与 OpenClaw 集成实践，称 ADP 支持 RAG、Workflow、Multi-Agent，并强调可快速接入企业微信、飞书、钉钉、QQ 等渠道。来源：https://cloud.tencent.com/developer/article/2680407
- [媒体] 新华网 2026-06-05 报道腾讯云 AI 产业应用大会，腾讯云发布“效率智能体工具集”，并升级 WorkBuddy 企业版 AI 工作台、ClawPro、腾讯云 ADP 等产品。该报道说明腾讯正在把 Agent 能力产品化、企业化。来源：https://www.news.cn/tech/20260605/e410f96e4f594ec0b92a4054faed70de/c.html
- [半官方/生态] Nexu 的 2026 接入教程显示，企业微信“智能机器人”API 模式可通过 Bot ID 和 Secret 以及长连接接入 Agent，并在企业微信中回复用户消息。它还提示组织内创建和使用智能机器人通常需要企业管理员权限。来源：https://nexu.io/zh/blog/how-to-connect-wecom-ai-agent

## 消息发送与 API 开放迹象

- `wecom-cli` 是今日最重要的开放迹象：它已经把消息读写能力包装成 Agent 可调用工具，但当前 README 同时写明了企业规模和场景限制。因此这不是“全面开放用户代发 API”，而是“在智能机器人 / CLI / Skills 框架下受控开放”。
- OpenClaw 插件 README 提到主动消息能力，且插件由 WeComTeam 维护。这需要继续确认其实际授权对象、目标范围、频率限制和是否只限机器人身份。
- 目前没有看到官方文档确认“外部联系人/客户群场景允许 Agent 或 API 以员工身份任意主动发消息”。这条仍应视为未开放。

## 未证实或低可信信号

- 社区和教程文章中常把“机器人可回复消息”“可接入群聊”“插件支持主动消息”写成接近完整客服能力，但这些描述不等价于官方开放“成员身份代发”或“外部联系人主动私聊”API。
- 企微智能机器人 API 模式的可用范围、是否能进外部群、是否能主动触达客户，仍需要以企业微信官方文档、机器人配置页和实际权限为准。
