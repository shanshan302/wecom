# Community And Event Sources

This file lists non-official or semi-official sources that are useful for tracking Enterprise WeChat / WeCom AI, Agent, MCP, intelligent bot, and message API signals.

## Excluded Routine Sources

Do not use these as routine daily sources:

- Bilibili
- YouTube
- Jike
- Weibo
- X

They may contain early rumors, but they are too noisy for the normal report. Use them only if a human explicitly asks for video or social monitoring.

- Tencent Cloud AI Industry Application Conference

## Event Scoring

When a report mentions an event, include:

- Date and city or online/offline format.
- Registration status when available.
- Agenda fit for WeCom, WeChat ecosystem, Tencent Cloud ADP, OpenClaw, MCP, AI Agent, intelligent bot, or message API topics.
- Speaker/source authority.
- Whether it is worth attending, watching replay, or only monitoring.
